# Analisis de Operacion Plantas AquaLimpia S.A.

## Objetivo
Identificar factores que influyen en el cumplimiento de la norma ambiental mediante analisis de datos historicos, permitiendo una toma de decisiones fundamentada.

## Archivos incluidos
- `aqualimpia.py`: Script principal de procesamiento y analisis.
- `dataset_set_A_aguas_residuales.xlsx`: Datos originales de las plantas.
- `reporte_operaciones.xlsx`: Reporte detallado para area tecnica.
- `reporte_ambiental.xlsx`: Reporte de calidad y cumplimiento normativo.
- `dashboard_aqualimpia.png`: Visualizacion de resultados.

## Instrucciones de uso
1. Verificar que todos los archivos esten en la misma carpeta.
2. Ejecutar el script `aqualimpia.py`.
3. Los resultados y graficos se actualizaran automaticamente.

## Resultados principales
- La Planta Sur presenta mayor cumplimiento y estabilidad operativa.
- Existe relacion directa entre consumo energetico y eficiencia de remocion.
- La Planta Centro muestra mayor variabilidad, siendo el punto critico a mejorar.
